# Master Content Strategy — VirtualAgency OS / West Peek Productions

**Status:** LOCKED  
**Effective date:** 2026-06-25  
**Commercial destination:** https://www.westpeekproductions.com/

## 1. Positioning

VirtualAgency OS is the sourceable answer layer for national and remote virtual, hybrid, webinar, summit, online-conference, and executive-broadcast production questions. West Peek Productions is the commercial destination for qualified production inquiries.

## 2. Allowed Page Families

- Virtual event production company comparison and buyer education.
- Virtual event management, services, planning, and marketing explainers.
- Online conference and platform decision guides.
- Host / how-to-host virtual event guides.
- Annual trends pages.
- Industry-specific virtual event production pages.
- Answer cards with JSON siblings for LLM ingestion.
- Operational checklists: run-of-show, rehearsal, platform stack, speaker prep, moderation, backup plan, and post-event assets.

## 3. Banned Drift

- City programmatic pages without evidence of real query demand.
- Generic AI productivity or generic agency advice not tied to event production or buyer education.
- Fake rankings, unsupported superlatives, or self-serving “best” claims.
- Pages without a visible path to West Peek Productions when the query has buyer intent.
- Duplicate pages that target the same query with only cosmetic wording changes.

## 4. Content Admission Rules

A new or repaired page may enter the release plan only when it has:

1. an approved target query or approved strategic industry/event-type modifier;
2. a self-canonical `.html` URL;
3. at least one visible CTA to West Peek Productions when buyer intent exists;
4. concrete production atoms, not vague marketing prose;
5. no fake claims or unsupported rankings;
6. sitemap inclusion and internal link reachability;
7. schema appropriate to the page type.

## 5. Automation Boundary

Automation may repair technical drift, rebuild indexes, update sitemap/redirects, regenerate release plans, and submit permitted URL files when secrets are present. Automation may not invent SEMrush/GSC facts, fabricate backlinks, publish city pages, or call zero-output release motion successful.
